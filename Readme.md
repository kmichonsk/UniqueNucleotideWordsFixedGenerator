# Unique Words Generator
Simple Java application, implementing non-optimal algorithm to generate fixed number (96) of unique 6 letter nucleotide words.
## How to run
### Dependencies
- Maven
- Java >= 11
### Steps
`mvn clean install`

`mvn exec:java -Dexec.mainClass="org.example.App"`
